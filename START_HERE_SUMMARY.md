## ✅ STEP-BY-STEP CHECKLIST

### PHASE 1: SETUP (5 minutes)

```
☐ Install VSCode
  → https://code.visualstudio.com

☐ Install Claude Extension in VSCode
  → Extensions → Search "Claude" → Install

☐ Get API Key
  → https://console.anthropic.com
  → Copy your API key

☐ Add API Key to Claude Extension
  → Click Claude icon → Settings → Paste key

☐ Create folder "RoyalSpicyPOS"
  → Open in VSCode

☐ Create 3 files in folder:
  → index.html
  → manifest.json
  → package.json
```

### PHASE 2: GENERATE CODE (5 minutes)

```
☐ Open Claude chat in VSCode
  → Click + for new chat

☐ Copy entire prompt from:
  → CLAUDE_PROMPT_EXACT.md

☐ Paste into Claude
  → Send message

☐ Wait for response
  → Claude generates all 3 files

☐ Claude will show:
  === FILE: index.html ===
  [long code]
  
  === FILE: manifest.json ===
  [code]
  
  === FILE: package.json ===
  [code]
```

### PHASE 3: PASTE CODE (5 minutes)

```
☐ Click index.html in VSCode
  → Right-click → Open
  → Paste ALL code from Claude's index.html
  → Save (Ctrl+S)

☐ Click manifest.json
  → Paste code from Claude's manifest.json
  → Save (Ctrl+S)

☐ Click package.json
  → Paste code from Claude's package.json
  → Save (Ctrl+S)

✅ All files now have code
```

### PHASE 4: TEST LOCALLY (2 minutes)

```
☐ Right-click index.html
  → "Open with Live Server"
  → OR drag to browser

☐ You should see:
  → Fullscreen app loads
  → No browser UI visible
  → Dashboard appears
  → 12 tables visible

☐ Quick test:
  → Click a table
  → Add an item
  → Bill calculates
  → ✅ Works!
```

### PHASE 5: DEPLOY TO VERCEL (5 minutes)

```
☐ Create GitHub account
  → https://github.com/signup

☐ Create repository
  → https://github.com/new
  → Name: RoyalSpicyPOS

☐ Upload 3 files to GitHub
  → Add file → Upload files
  → Drag files to GitHub
  → Commit

☐ Create Vercel account
  → https://vercel.com/signup

☐ Deploy to Vercel
  → Click "Add New Project"
  → Select GitHub repo
  → Click Deploy
  → Wait 30 seconds

☐ ✅ GET LIVE URL!
  → Example: https://royalspicypos.vercel.app
```

### PHASE 6: INSTALL ON DEVICES (2 minutes)

```
IPAD (Safari):
☐ Open Safari
☐ Go to Vercel URL

ANDROID (Chrome):
☐ Open Chrome

☐ Menu → Install app
☐ ✅ App installed!

TABLETS:
☐ Same as iPad or Android process
```

## 🆘 IF SOMETHING GOES WRONG

```
Claude not working?
→ Check API key in VSCode settings
→ Try restarting VSCode
→ Generate code again

Code not working?
→ Check all 3 files are saved
→ Open in Chrome browser
→ Check console for errors (F12)

Vercel deployment fails?
→ Make sure package.json is valid
→ Check all files in root folder
→ Try redeploying

Bluetooth printer not printing?
→ Make sure printer is paired
→ Use Chrome or Edge
→ Check printer is powered on
→ Check printer has paper
```
